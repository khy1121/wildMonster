## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`


향후 개발 방향 제안

퀘스트 시스템 완성: 퀘스트를 일일/주간/공통으로 분류하고, 달성 조건과 보상을 설정합니다. 퀘스트 달성 시 팝업과 보상 지급을 구현하고, 퀘스트 로그에 달성 여부를 기록합니다. 일일 퀘스트는 매일 갱신되며 재롤 기능을 통해 사용자 선택을 허용할 수 있습니다
nerdexp.com
nerdexp.com

진화 및 레벨 시스템 확장: 테이머 최대 레벨(50)과 와일더 최대 레벨(80)에 맞춰 경험치 곡선을 조정합니다. 레벨이 높아질수록 레벨업에 필요한 경험치가 기하급수적으로 증가하도록 설계해 도전감을 높입니다. 동시에 퀘스트 보상(경험치, 골드, 아이템 등)을 통해 성장 속도를 조절합니다. 분기 진화는 이미 구현된 예시를 확장하여, 각 진화 경로별 고유 스킬과 스탯 상승을 부여합니다.

3D UI/그래픽 강화: enable3d 또는 Three.js로 구현된 3D 오브젝트와 파티클을 UI 요소에 통합하여 시각적 매력을 높입니다. 예를 들어 홈 화면에 회전하는 3D 보석 모델을 띄우고, 스킬 버튼 위에 입체 파티클 이펙트를 추가하여 인터페이스에 생동감을 부여합니다. Phaser의 파티클 시스템으로는 불꽃 이펙트 등을 구현하여 스킬 사용 시 연출에 활용할 수 있습니다
phaser.io
